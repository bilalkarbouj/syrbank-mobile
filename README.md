# SYRBANK – Next Generation Payment App

SYRBANK is a modern mobile payment application built with **Flutter** that enables users to manage their finances quickly, securely, and intuitively.

The application provides essential digital banking features such as **money transfer, QR payments, real-time balance tracking, and transaction history**, all within a clean and user-friendly interface.

This project was developed as a **portfolio and learning project** to demonstrate modern mobile banking architecture and scalable app design.

# 🚀 Features

• 💳 **Digital Wallet**  
View and manage your account balance in real time.

• 💸 **Instant Money Transfer**  
Send money between accounts quickly and securely.

• 📷 **QR Code Payments**  
Pay or receive payments instantly using QR codes.

• 📊 **Transaction History**  
Track all incoming and outgoing transactions.

• 🔐 **Secure Authentication**  
Login and account management system.

• ⚡ **Fast and Modern UI**  
Smooth user experience with a clean mobile interface.

• 🌙 **Dark Mode Support**  
User-friendly dark interface option.

• 🔔 **Notifications System**  
Receive updates for transactions and account activities.

• ⚙️ **Profile & Settings Management**  
Manage personal information and security settings.
